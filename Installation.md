# Roomiez

## A website to connect people looking for a roommate!
### Installation Instructions
#### Requirements
```
1) Install InteliJ IDEA
2) Install MySQL
3) Working web browser
```
#### Installation
```
1) Git clone https://githome.com/WSU-4110/Roomiez.git
2) Run MySQL
3) Start the Spring Server
4) Run RoomiezApp
```
